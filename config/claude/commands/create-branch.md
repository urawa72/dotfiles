# Create a Git Branch

Create a Git branch with an appropriate branch name.

## Rules

- Use `git status` and `git diff` to understand the changes
- Create the appropriate branch
- Follow [Conventional Branch](https://conventional-branch.github.io/) naming conventions for branch names

## Example

```
feature/admin-user-role-edit-invite-form
```
